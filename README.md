# Nivara

## Core Features

- User Login

- Health Monitoring Dashboard – tracks heart rate, blood pressure, steps, sleep, and calories with daily, weekly, and monthly progress graphs

- AI Mental Health Chatbot – provides stress relief, mindfulness conversations, motivational support through NLP, and suggests ayurvedic or nature-based lifestyle tips (non-medical)

## Tech Stack

- **Frontend:** Next.js with React, TypeScript, and Tailwind CSS
- **Backend:** Next.js API Routes (serverless functions)
- **AI Integration:** Gemini NLP model for conversational AI chatbot

## Project Overview

**Project Name:** Nivara – Smart Health & Wellness Companion

Nivara is a conceptual digital wellness platform designed to promote holistic health by combining technology with lifestyle insights. It allows users to monitor basic health parameters, view progress trends, and engage in supportive mental wellness interactions.

### Key Objectives

- Enable users to create secure accounts and track personalized wellness data

- Provide a dashboard displaying health metrics with visual progress reports

- Offer a conversational AI assistant focused on emotional support and lifestyle recommendations

- Encourage healthy living through non-clinical, natural, and balanced approaches

## Purpose

The primary goal of Nivara is to help individuals maintain mental and physical balance through data-driven insights and supportive guidance. It aims to promote awareness and daily wellness, not to replace professional healthcare.
